# MCBEEditor 玩家地图图层与整体重命名

- 软件整体名称由旧名称统一改为 `MCBEEditor`，版本继续固定为 **1.0.0 (100)**。
- 同步修改 Xcode 工程名、Target、Scheme、测试 Target、Bundle ID、产品名、启动页、应用显示名称、桥接头文件、GitHub Actions、构建脚本、IPA/Artifact 名称、应用支持目录、队列标识、错误类型与导出文件名。
- NBT JSON 新格式标识改为 `mcbeeditor-nbt-json`，NBT 剪贴板类型改为 `com.wzn.mcbeeditor.*`；旧版格式标识仍可导入，避免历史导出文件和剪贴板内容失效。
- 地图对象图层增加默认开启的“显示玩家”，位于“显示实体”上方；“显示出生点”移动到“显示方块实体”下方。
- 本地玩家绘制为黄色五角星，在线玩家绘制为蓝色五角星；位置直接来自玩家 NBT，而不是普通实体扫描。
- 点按玩家五角星会显示玩家 identifier、UniqueID、类型、维度、坐标与来源，并提供编辑 NBT、定位和复制坐标操作。
- 地表、高度等六种地图渲染模式不再跨世界会话保存；退出存档后删除旧持久化值，下次打开恢复默认地表模式。
- 地图中心与缩放仍只保留在当前打开存档的会话内，切换栏目或维度不会意外重置。
- 新增 `Scripts/test_mcbeeditor_map_player_rename.sh`，检查图层顺序、玩家颜色、点击菜单、会话级渲染模式与全项目重命名字段。
